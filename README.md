# Finanças++

Projeto "Finanças++" desenvolvido para a disciplina Linguagem Script para Web (LSW), do Professor Fábio Bala.

O projeto é voltado para o controle e gerenciamento financeiro pessoal, permitindo o registro de receitas e despesas, categorização de gastos e geração de relatórios básicos para auxiliar no planejamento financeiro individual.
